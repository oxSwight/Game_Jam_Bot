# Политика конфиденциальности и правила участия

**Версия 1 · действует с 7 июля 2026**

Этот документ описывает, какие данные собирает регистрационный бот закрытой
игровой группы (далее - «бот»), зачем они нужны, кто имеет к ним доступ и как
их удалить. Краткая версия этого текста показывается в боте на шаге согласия;
нажатие кнопки «Принимаю условия» означает согласие с настоящей политикой.

---

## 1. Какие данные мы собираем

При подаче заявки бот сохраняет:

| Данные | Источник |
|---|---|
| Telegram ID и username | автоматически из вашего аккаунта Telegram |
| Никнейм (отображаемое имя) | вводите вручную |
| Email | вводите вручную |
| Ответы анкеты: категория, роли, опыт, движок, инструменты, мотивация | выбираете в анкете |
| Язык интерфейса (RU/EN) | ваш выбор или язык клиента Telegram |
| Факт членства в группе (в группе / не в группе) | события Telegram о входе/выходе |
| Служебный журнал действий по заявке (подача, одобрение, отклонение) | автоматически |

Бот **не** собирает: номер телефона, геолокацию, содержимое ваших сообщений вне
анкеты, контакты, данные платёжных средств.

## 2. Зачем эти данные нужны (цели обработки)

- рассмотрение вашей заявки модераторами (ник, email и ответы анкеты видны
  модераторам в карточке заявки);
- выдача персонального приглашения в закрытую группу и подтверждение, что в
  группу входит именно одобренный участник;
- связь с вами по вопросам участия (email - резервный канал связи);
- обезличенная статистика по категориям и опыту (без имён и контактов).

Данные **не** используются для рекламы и **не** передаются третьим лицам.

## 3. Кто имеет доступ

Только администраторы закрытой группы (список фиксирован конфигурацией бота).
Технически данные хранятся в базе данных проекта; доступ к серверу есть только
у администраторов проекта.

## 4. Сколько данные хранятся

- Пока ваша заявка активна или вы состоите в группе.
- Вы можете **в любой момент удалить все свои данные** командой `/withdraw` в
  боте: удаляются заявка (включая отклонённые), никнейм, email, username,
  языковая настройка и связанный журнал действий. Это необратимо; после
  удаления можно зарегистрироваться заново «с нуля».
- Выход из группы сам по себе **не** удаляет данные (бот помнит вернувшихся
  игроков) - для полного удаления используйте `/withdraw`.

## 5. Ваши права

- Посмотреть, что о вас хранится: команда `/status`.
- Исправить данные: команда `/edit` (ник, email, навыки).
- Удалить данные: команда `/withdraw`.
- Вопросы по данным: напишите любому администратору группы.

## 6. Правила участия

1. Это MVP-тест сервиса: возможны сбои, о них можно сообщать администраторам.
2. Заявки проверяются вручную; решение модератора окончательное, но после
   отклонения можно подать новую заявку.
3. В анкете запрещены ссылки и спам - такая заявка сбрасывается автоматически.
4. Пригласительная ссылка персональная: вход в группу подтверждается ботом
   только для одобренного участника, передавать ссылку другим бессмысленно.
5. Будьте готовы по запросу модератора подтвердить заявленные навыки
   (портфолио / примеры работ).
6. В группе действует уважительное общение; нарушение - исключение из группы.

## 7. Изменения политики

При изменении политики её версия увеличивается, и бот запросит согласие заново
при следующей подаче заявки. Факт и версия вашего согласия фиксируются в
журнале заявки.
